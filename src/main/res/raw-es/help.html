Introducción

Un sintonizador del acordeón del androide con el estroboscópico y las notas múltiples.

Utilizando

Sostenga el teléfono/tableta cerca del instrumento, toque una nota y observe la pantalla. En el retrato, la aplicación muestra un alcance con la señal de entrada, un espectro que muestra la señal procesada, la pantalla principal con el valor de la nota, el error de centavos, el error de frecuencia, etc., una pantalla estroboscópica y una pantalla del medidor que muestra el error de centavos. En el paisaje, la aplicación sólo muestra la pantalla principal y el medidor. Hay un indicador de señal en la barra de acción que es útil en modo horizontal. El uso de los cambios de configuración táctiles a continuación hace que aparezca un breve mensaje de notificación durante unos segundos.

Osciloscopio

Toque el osciloscopio para agregar un filtro de paso bajo de entrada para que sea más fácil afinar las cañas de bajos eliminando los armónicos. Toque de nuevo para eliminarlo. Los colores se han elegido tanto para la claridad, y para que "retro" se ven como una pieza real de hardware.

Espectro

Toque el espectro para acercar la nota actual. La pantalla puede mostrar hasta ocho cañas. Toque de nuevo para alejar el zoom. Toque y mantenga presionado para agregar una función de downsampling para ayudar con las cañas de bajo. Esto puede producir resultados espurios. Toque y mantenga presionado de nuevo para quitar.

Pantalla

Toque la pantalla para bloquear los resultados actuales. Toque de nuevo para quitar el bloqueo. El bloqueo de pantalla no se conserva si se gira la pantalla. Mantenga pulsada la pantalla para cambiar entre mostrar el resultado más cercano a la nota de referencia y hasta ocho resultados. Esta configuración no se recordará la próxima vez que utilice la aplicación.

Estroboscópico

Toque el estroboscópico para encenderlo y apagarlo. Los colores estroboscópicos son configurables en los ajustes. Mantén presionado para cambiar el tema.

Medidor

El medidor muestra el error de los centavos hasta más o menos 50 centavos. Toque el medidor para copiar los resultados en el portapapeles. Manténgalo presionado para evitar que la pantalla se apague. Toque y mantenga presionado de nuevo para permitir que la pantalla se apague.

Barra de estado

La barra de estado muestra la frecuencia de muestreo de entrada y el estado de los diversos ajustes.

Ajustes

Todas las configuraciones se conservan excepto la opción de múltiples notas y se recordarán la próxima vez que utilice la aplicación.

Entrada

Se puede cambiar la fuente de entrada. Si se selecciona un origen que no está presente en el dispositivo, aparecerá un mensaje de error. Seleccione la fuente predeterminada o el micrófono. En teoría, puede utilizar esta función para sintonizar un instrumento en el otro extremo de una llamada telefónica.

Filtro de audio

El elemento de filtro de audio controla un filtro de paso bajo en la entrada de audio antes del procesamiento.

Bajamar

El elemento bajar la muestra controla una función de downsampling en la entrada de audio durante el procesamiento para ayudar con las cañas de bajos. Use con precaución, puede generar resultados espurios.

Pantalla

Mantenga la pantalla encendida

El elemento de pantalla detiene la pantalla apagándose.

Zoom espectral

El elemento de zoom de espectro controla la función de zoom de la visualización del espectro.

Mostrar varias notas

El elemento de varias notas controla la visualización de más de un resultado en la pantalla. Esta configuración no se conserva.

Estroboscópico

Estroboscópico

El elemento estroboscópico controla la visualización estroboscópica.

Colores del estroboscópico

Si la luz estroboscópica está habilitada, hay una opción para cambiar los colores. Hay tres ajustes predefinidos y un ajuste personalizado. Si se selecciona la configuración personalizada, los dos colores se pueden cambiar individualmente. Toque el círculo de color exterior para seleccionar un color. Toque el círculo de color central para cada color para ver cómo se verá a la izquierda..

Referencia

La frecuencia de referencia para 'A' se puede cambiar en un rango limitado. Esto se recordará, así que cambia de nuevo a 440Hz antes de salir de la aplicación.

El elemento de transposición permite la transposición de la visualización de notas. Esto se recordará, así que cambie de nuevo si ya no es necesario.

Tema

El elemento tema oscuropermite el uso del tema oscuro.
